# MedTimer

**About the App**

MedTimer is a medicine tracking app that helps users, often the elderly, but open to anyone who needs help staying consistent with their medications. Reminders, visual tracking tools, and motivational quotes encourage the user to make this a positive habit. The apps simplicity makes it accessible to anyone, regardless of age or tech experience, ensuring easy navigation. This app can be utilized to organize medicines, keep track of daily doses, and measure consistancy all in one place.

**Key Features**
* Index of managed medicines: This index contains a list of all the users medications. The user can add information about the medicine, the dosage, timings, and any additional notes. It is also easy to remove and add entries into a user's index.
  
* Daily Checklist: The daily checklist contains all the medicines needed to be taken on a particular day. The user checks off all the medicines already taken, creating a habit of staying on track. The app will also change the checklist according to the user's actions. It displays missing, taken, or upcoming based on the timings of the dosage.
  
* Weekly Adherence with rewards: Weekly adherence can be monitored through a weekly growth bar. Users can earn small rewards or badges for staying consistent, engaging the user to make tracking their medications part of their daily routine.
  
* Weekly Reports: A ready to download pdf of the user's weekly reports consists of an easy to follow summary of their weekly consistancy with their medication. This report can be shared with the user's caregiver or doctor for them to efficiently keep track of their patients medication data.
